☁️ Paramètres Cloud (Scaleway)

Connectez ListKontrol à votre backend Scaleway pour sauvegarder vos données dans le cloud (Varsovie, UE). Les données restent aussi en local pour une utilisation hors ligne.

📍 Datacenter: Varsovie, Pologne (UE)
🔒 Chiffrement: HTTPS + token d'authentification
📱 Mode hors ligne: Les données locales restent toujours accessibles
